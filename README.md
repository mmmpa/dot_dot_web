# ショートカット

## マウス操作

|キー|左クリック|右クリック|
|:---|:---|:---|
|-|カラー1描画|カラー2描画|
|Ctrl|カラー1スポイト|カラー2スポイト|
|Shift|選択|選択解除|
|Ctrl+Shift|矩形選択|-|

## キー操作

|キー|動作|
|:---|:---|
|Ctrl+C|選択範囲をコピー|
|Ctrl+X|選択範囲をカット|
|Ctrl+V|クリップボードからペースト|
|カーソル|ペースト時、ペースト部位を移動|
|Ctrl+カーソル|選択範囲をカットペーストして移動|
|G|グリッドの表示非表示|
